# Bp核心库
## Bp
* 全局初始化所有配置
* 全局清理

## BpLib
主要管理所有基础模块

## BpObj
所有蓝图节点的基类, 定义输入输出, 节点类型
virtual void Run();
virtual void Logic() = 0;

## BpNode
节点: 
* 函数指针节点
* 分支结构节点
    * if..else..
    * while
    * seq
* 输入输出节点
* 变量节点

## BpGraph
图:可分为组织模块,或者一个可执行的图(需要标记一个图是组织模块还是可执行类型)
* 组织模块
    * 输入输出
    * 根据输入输出显示输入输出节点
        * 输入必须有一个或多个flow类型的pin
* 可执行的图
    * 加载配置文件
    * 保存配置文件
    * 设置多个执行入口节点(顺序执行)
        * 显示入口执行节点
    * 创建修改删除变量
